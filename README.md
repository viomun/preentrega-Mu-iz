# Proyecto de Automatizacion 
- Talento Tech
Automatización de pruebas funcionales sobre la pagina "https://www.saucedemo.com/", utilizando ** Python**, ** Selenium** y ** Pytest**
El proyecto incluye:
-Prueba de login exitoso
-Prueba de navegación del catalogo
-Prueba del carrito

## Requisitos previos
-Instalar Python
-Instalar Pytest
-Instalar Selenium
-Instalar Webdriver
